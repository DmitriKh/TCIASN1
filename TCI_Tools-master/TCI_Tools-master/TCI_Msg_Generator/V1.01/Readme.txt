1-	To install, double-click on the setup.exe file;
2-	The installer will create a shortcut on your desktop;
3-	The installer will create a sub-directory “Danlaw” in your c:\users\public directory;
4-	The installer will copy some the xml files (used with WAVE Services)and a text file to the directory that you created in step 3;
5-	Edit the SutIPv4Address.txt file to have the IP address of your device under test;
6-	With the exception of the WAVE Service in 1609.3, the GUI is self-explanatory and easy to use.

The software exercises every message defined in the ASN.1 files, except for the shutdown and restart.
